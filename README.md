# Green Honey

Different languages have different ways of describing colour. Using English and Chinese as examples, this web project visualises the difference between the two languages and their use of colour names.

## Technologies used

- Gulp
- Jade
- D3
- jQuery
- Bootstrap
- Stylus
- Node.js
- LiveScript
